# 全局翻译规则

| 英文术语                        | 推荐译法         | 备注                         |
| --------------------------- | ------------ | -------------------------- |
| reinforcement learning      | 强化学习 / RL    | 标题或概念介绍可译；技术上下文可保留 RL      |
| deep reinforcement learning | 深度强化学习 / DRL | 同上                         |
| policy                      | policy       | 不强制译为“策略”；必要时写作 policy（策略） |
| reward                      | reward       | 不强制译为“奖励”；必要时写作 reward（奖励） |
| observation                 | observation  | 不强制译为“观测”                  |
| action                      | action       | 不强制译为“动作”                  |
| state                       | state        | 不强制译为“状态”                  |
| environment                 | environment  | RL 语境下优先保留英文               |
| agent                       | agent        | 不译为“智能体”，除非中文上下文需要         |
| task                        | task         | 不强制译为“任务”                  |
| training                    | training     | 不强制译为“训练”                  |
| inference                   | inference    | 不强制译为“推理”                  |
| deployment                  | deployment   | 不强制译为“部署”                  |
